# Quantitative Trading Radar Terminal

An interactive terminal-based stock analysis tool that provides real-time technical insights using Moving Averages, Bollinger Bands, RSI, Volume analysis, and algorithmic trading signals.

## Features

- Moving Averages (20/50/100/200 SMA)
- Bollinger Bands (20-period) with Bandwidth
- 14-period RSI
- Volume Analysis
- Smart Trading Signals with scoring system
- Clean professional terminal UI

## Installation

```bash
pip install pandas yfinance
```
